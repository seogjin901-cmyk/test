# Tetris+ (커스텀 테트리스)

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

Tetris+는 HTML, CSS, JavaScript로 만든 커스텀 테트리스 게임입니다.  
기본 테트리스 기능을 기반으로 **점수 시스템, 블록 윤곽선, 줄 제거 점수, 블록 고정 점수, 속도 증가** 등의 다양한 기능이 추가되었습니다.

---

## 🎮 게임 특징

- 블록 종류: I, J, L, O, S, T, Z  
- 블록 설치 가능 영역 **오른쪽으로 2칸 확장** (기본 12칸)  
- 겉 윤곽선 유지 + 내부 윤곽선 얇고 연한 점선으로 표시  
- 한 줄 완성 시 블록 제거 + **점수 100점**  
- 블록 고정 시 점수 지급 (기본 50점, 속도 증가 시 +20점)  
- 블록 고정 후 속도 점진적으로 증가 (1000점 단위)  
- HOLD 기능: Enter 키로 블록 저장/교체  
- 하드 드롭: 스페이스바로 블록 즉시 낙하
